**Weather Forecast App (Django)**

This is a simple Weather Forecast Web App built using Django and a free weather API.

Users can search any city (for example: Paris) and see the weather details.

Search weather by city name

**How to Run**
git clone https://github.com/your-username/weather-api-django.git
cd weather-api-django
pip install
python manage.py runserver


**Open in browser:**

http://127.0.0.1:8000/
 **API Note**

Add your free weather API key in the project settings.
